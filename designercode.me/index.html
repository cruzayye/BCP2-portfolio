Light Intro to Backend Development

Long story short Backend Development is working with databases and dealing with sending and receiving data(http requests), but if your want a bit more info please continue reading...
Before this year, I knew that when it came to web development, there are 3 types of developers: backend developers, front end developers and fullstack developers. While front end devs focus on making sites look pretty and everything HTML, CSS, and Javascript. Backend devs deal with JS algorithms as well as working with servers, including retrieving users names passwords and information that the website handles. This leads to us to the topic of HTTP Methods aka CRUD, which is short for 'create', read, update and delete. okay... but how is this used? Let's use Instagram for example since that's what us millennials are into. When we open up the app and go to the home page to look at everyone's post we tapping into Instagram's database and asking it to GET data for us, which means that the developers have to make a function that lists all of the new posts that our friends have posted when we click the home icon. Have you ever thought about what kind of method is used to 'upload' a picture? Well, when uploading or creating any kind of content for the world to see we use a POST method to send new information to the server. What about to delete? well, you guessed it DELETE. Another commonly used method would be PATCH, or PUT which wouldn't be used for a picture but for a comment. For example, you might have made typo; sometimes we might spell a word so wrong on our smartphone that autocorrect isn't even able to correct it. If you're considering getting started with Backend development and be able to dabble with these CRUD methods You're going to have to look into Node. This link is a deep introductory to Node JS is the javascript framework that lets us do backend development. Once you get done with that check out these articles as well.. or just skip ahead and get a preview of what to come!

Want to Learn Code??

So you want to learn about code huh? If you enjoy not having an idea about what you’re doing and your pretty decent at maneuvering your way through google to find your answers, then you chose the right field. If you have no Idea what type of coding you want to dive into, figure out what technology you want to learn...Do you want to make video games for Nintendo or Playstation? Then you would want to look into c++, c# and java. Want to learn about data management, working with tables and using that information in a unique way such as portraying stats? SQL or Python are whee its at. If you’re like me and you're looking into designing websites and web applications then you’re going to start with html/css, then move onto javascrtipt, which makes websites more robust and intuitive. Since I’m a front end developer Heres my secret:

  1. Learn the basics of HTML/CSS for free at khan Academy Look for the section about creating webpages.
  2. Make a website. Start with a simple gallery of images with a 3 rows and 3 columns or something better you can come up with. There are several tutorials out there, use your google skills to find the right one.
  3. Learn JS and practice everyday. There are so many websites as resources out there that its hard to choose. Check out these 3 sites. What's awesome about Udemy is that they provide full on courses and have sales all the time where the course turns out to be less than $15.00. Theres a lot of hype around freecodecamp as well and I sometimes use it as a reference.
    • Udemy
    • Free Code Camp
    • Khan Academy

Toughest part about learning on your own is discipline and holding yourself accountable. No one’s going to tell you what you have to do as far as staying consistent so it’s easier to just tell yourself that you’ll get to it eventually, but Soon enough eventually becomes never. If this is your issue check to see if theirs an in person bootcamp in your city. Being around coders will help you make connections and push you further. Personally I don't think I would have learned as much online because my lack of discipline, but thats just me, chances are you're a lot more self motiated than me. Don’t get discouraged to join an online bootcamp, the right one will have a course that gets you to build projects that lead into a plehora amount of knowledge. BUT PLEASE DO YOUR RESEARCH because not all of them are going to give you the tools and knowledge to prep you for a job. If I never joined a bootcamp I would have never considered learning github, SQL, the command line and practicing code on sites like Codewars and Codesignal. Long story short DO YOUR RESEARCH, commit and code everyday!
